# «Сервисный центр»

Информационная система для автоматизации учета заказов, складских запасов и предоставляемых услуг в мастерской по ремонту цифровой и бытовой техники. Приложение построено на базе архитектуры Windows Forms и использует локальную базу данных для обеспечения высокой скорости работы.

## Основные возможности

### 📋 Управление заказами
* **Оформление и редактирование:** Регистрация клиентов, фиксация контактных данных, типа устройства, модели и серийного номера.
* **Гибкая классификация:** Встроенный упорядоченный справочник из 35+ типов устройств (от мобильных телефонов до серверов и кофемашин) с возможностью выбора варианта «Другое».
* **Учет состояний:** Отслеживание этапов ремонта по статусам (*Принят, В работе, Готов, Выдан*).
* **Динамическая корзина:** Интеграция выполненных услуг и списанных деталей прямо в карточку заказа с автоматическим расчетом итоговой стоимости в режиме реального времени.

### 📦 Складской учет (Запчасти)
* **Контроль остатков:** Учет деталей по наименованиям, уникальным заводским артикулам, количеству и стоимости.
* **Умное списание:** Автоматическое уменьшение количества компонентов на складе при добавлении их в заказ и мгновенный возврат на склад при удалении детали из корзины до сохранения.
* **Защита данных:** Система выдает предупреждение с запросом на подтверждение при попытке безвозвратного удаления позиции со склада.

### 💰 Прейскурант услуг
* **Каталог работ:** Ведение списка предоставляемых услуг и их фиксированной стоимости.

### 🔍 Поиск и фильтрация
* Мгновенный поиск информации по мере ввода текста на каждой из основных вкладок (поиск заказов по клиенту/модели, поиск запчастей по названию/артикулу, поиск услуг по наименованию).

## 🛠 Технологический стек

* **Язык программирования:** C# (.NET Framework / .NET Core)
* **Интерфейс:** Windows Forms (WinForms)
* **База данных:** SQLite (локальный файл, не требующий установки отдельного сервера СУБД)

---

## 🖥️ Требования к компьютеру

* **Операционная система:** Windows 10 / Windows 11 (32-bit или 64-bit).
* **Процессор:** С частотой 1.6 ГГц или выше (двухъядерный).
* **Оперативная память (RAM):** Минимум 2 ГБ (рекомендуется 4 ГБ для плавной работы с большими таблицами).
* **Свободное место на диске:** Не менее 50 МБ (основной объем будет занимать файл базы данных SQLite по мере накопления заказов).
* **Разрешение экрана:** Минимальное разрешение 1024x768 пикселей (для корректного отображения всех полей формы редактирования заказа).
